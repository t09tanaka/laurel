# Claude workspace

This directory holds project-local Claude Code customizations.

- `agents/` — project-local subagents. `ghost-helper.md` is the one to invoke
  when implementing or debugging a Ghost helper; it knows the conventions in
  `src/render/helpers/`.

Settings for permissions, hooks, and so on live in `.claude/settings.json` (or
`settings.local.json` if you want personal overrides). They are intentionally
not auto-generated by the assistant — set them yourself with `/permissions` or
by hand.
